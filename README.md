# simple_lightbox.js

<h4>轻量</h4>
这是一个用纯原生js编写的一个lightbox插件，无需引用过多的css，你仅仅需要引入这个js文件即可

<!-- ![image](https://github.com/flouthoc/Uglipop.js/blob/master/shot.png) -->

<h3>如何调用</h3>
```html
  <div class="test">
    <img src="1.jpg">
    <img src="2.jpg">
  </div>
```
```javascript
   <script>
      previewImage('.test')
   </script>        

```
